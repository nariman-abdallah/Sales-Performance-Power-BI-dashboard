# Sales-Performance-Power-BI-dashboard
Project Overview
This project analyzes sales performance using Power BI to provide executive-level KPI reporting and business intelligence insights.

Dataset:
-Sales transactions dataset
-Includes revenue, profit, orders, segment, category, region, and shipping mode

Key KPIs:
-Total Sales: 2.30M
-Total Profit: 286.40K
-Total Orders: 5K
-Profit Margin: 12.47%

Tools Used:
-Power BI
-Power Query (ETL)
-DAX Measures
-Data Modeling
-Interactive Filtering
-Figma

Key Insights:
-Seasonal peaks identified in monthly sales trends.
-Technology category generated highest revenue.
-Regional analysis revealed performance differences across states.

Business Impact:
Provides strategic sales insights to support revenue optimization and performance monitoring.
